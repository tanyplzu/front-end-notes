# pnpm

```shell
# 安装
npm install -g pnpm

# 淘宝镜像
pnpm config set registry https://registry.npm.taobao.org/

# 设置存储路径
pnpm config set store-dir /path/to/.pnpm-store

# 查看全局安装信息
pnpm list -g

# 新增
pnpm add xxx

# 删除
pnpm remove xxx

# 更新
pnpm update xxx
```

[pnpm](https://pnpm.io/zh/cli/setup)
