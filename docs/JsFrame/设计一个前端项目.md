## 使用一致的代码开发规范

可以使用一些工具来确保代码符合规范：

- 使用 Eslint 检测代码规范；
- 使用 Prettier 自动化格式代码；
- 使用 Git Commit Hooks 拒绝不符合规范的代码提交；
- 使用流水线检测出不规范的代码，并拒绝合入主干分支；
